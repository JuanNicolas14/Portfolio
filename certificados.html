Cursos Realizados

Frontend Developer Use HTML Y CSS para dar estilos al front de Platzi Video, una aplicación para ver videos en línea. Diseñe el front del home, la pantalla de registro y el login.

Curso Basico de JavaScript En este curso hecho en platzi, aprendi de temas como Variables, Funciones y sintaxis, Operadores, Condicionales, Arrays, Loops, objetos y metodos.

Curso practico de HTML y CSS En este curso práctico cree una copia de la página de inicio de google por medio de HTML y CSS.

Curso de Desarrollo web En este curso aprendi acerca del DOM, etiquetas, la estructura de los sitios web, aprendí de atributos y formularios HTML, tambien de selectores, propiedades y valores CSS, aprendi tambien acerca de bordes, margenes y rellenos, el modelo de caja y tipos de display, flexbox, entre otros.

Curso de Fundamentos de JavaScript Apliqué todos los conceptos básicos y fundamentos del lenguaje para desarrollar un juego que consiste en clickear la misma secuencia que se ilumina y en el orden correcto.

Curso de Responsive Design En este curso aprendí acerca de las medidas relativas utiles en el responsive design, media queries y las formas de incluirlo, y tambien sobre CSS positions.

Curso de CSS grid Aprendi a definir columnas, filas, tambien a definir areas de contenido y alinearlo, en este curso hice el layout de instagram y el landing de los cursos de platzi.

Curso de POO Aprendi mas acerca de los objetos, conceptos como abstraccion, modularidad, la herencia, capsulamiento, poliformismo,entre otros. Tambien prendi acerca del metodo constructor y su sintaxis.
